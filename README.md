# Sistema de Portaria — Sol do Vale

Sistema de controle de acesso e registro de visitantes para o condomínio Sol do Vale.

## Funcionalidades

- Registro de entrada e saída de visitantes
- Controle de acesso de moradores
- Histórico de movimentações
- Relatórios de ocupação

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/thiagohgmolino/sistema-de-portaria.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd sistema-de-portaria
   ```
3. Siga as instruções de configuração (em breve).
